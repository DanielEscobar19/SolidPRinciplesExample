Repositorio para la tarea sobre los princios SOLID de la práctica supervisada. 

El código fue creado con Visual Studio 2022 y .NET 6.

# Ejecución

Abrir el proyecto con visual studio y darle a la opción de ejecutar el archivo llamado **MainProgram.cs**

Otra opción es abrir la carpeta llamada **SolidPrinciplesExample** y ejectura el siguiente comando:

	dotnet run MainProgram.cs
